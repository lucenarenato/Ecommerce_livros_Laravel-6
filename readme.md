## Passos para utilizar

### composer update
### Configurar o arquivo .env com as configurações do banco
### php artisan migrate
### php artisan db:seed

```sh
    composer install
    php artisan migrate --seed
    php artisan key:generate
    php artisan serve --host=0.0.0.0 --port=8001
```
